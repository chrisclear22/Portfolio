BPA Competition Site

BPA Competition Site

This project was done in collaboration with two others. This site was created for the Business Professionals of America's competition. So far we have placed 1st in the state of Oklahoma and are preparing for nationals in Anaheim, California.

For this site I like to brag because I coded 95% of the entire site. My two team mates were in charge of design and project management. My job for the site was pure development. This site includes heavy PHP/MySQL, CSS, HTML, as well as two APIs we are using for Google maps and pulling in weather information for each destination. This site was also built using the Bootstrap framework.

Discography

Discography Project

This is a project I worked on in class. For this poject I was tasked with picking a band and creating a discography site for that particular band. I choose Three Days Grace simply on a whim. This project was one of the very first sites I have had to create from scratch.

While working on this site I highly increased my HTML and CSS skills. For this project I also had to create a Github repository for future reference. This site was then altered by a fellow student aiming on the design and layout of the site.

PHP Project
